# UI Foundry Textarea Component Generator

Standalone visual textarea generator built with Next.js.

This app is a focused multiline-field studio for comments, messages, notes, descriptions, and larger text-entry surfaces that can be edited visually and exported as code.

## Product summary

This generator is designed for:

- designers building contact forms, support flows, dashboards, and messaging UI
- developers who want editable multiline-field styling without rebuilding the component manually
- freelancers and agencies preparing branded form systems for client projects
- buyers who want a standalone textarea tool consistent with the UI Foundry suite

## What the product includes

- section-based editing controls
- live preview panel
- code view with copy support
- file export support
- searchable preset library
- generated premium preset catalog with filters, paging, and surprise-me apply
- richer platform-attribute controls including `id`, `title`, `tabIndex`, `dir`, `lang`, `inputMode`, `enterKeyHint`, `autoCapitalize`, `autoCorrect`, `minRows`, and `maxRows`
- deeper multiline editing with dedicated field-attribute and writing-mode sections
- preview/export-safe description and success messaging, structured supporting-copy rules, and quick long-form variant switching
- undo and redo controls
- self-contained standalone app structure

## Editing coverage

The editor currently includes these control areas:

- Presets
- Basics
- Field
- Styling
- Typography
- Writing
- States
- Effects
- Labels
- A11y

These sections cover field structure, platform attributes, writing modes, styling, typography, states, visual effects, label and supporting-message behavior, and a broader long-form input attribute surface for production-ready textareas.

## Export workflow

The textarea preview and code output are based on the same current state so the exported result follows the visual editor.

## Why this product is useful

This is more than a plain textarea snippet. It gives the buyer a configurable multiline-input studio for product-ready form surfaces.
